Smooth Terrain Loading is a Minecraft resource pack that loads in terrain in a world smoothly by using Vanilla core shaders. No mods required!

## Features

- **Terrain Loading**: Smoothly loads in terrain as you explore the world, eliminating chunks suddenly popping into view.
- **Vanilla Compatibility**: Works in Vanilla, without any mods or additional tools.
- **Lightweight**: A tiny resource pack that is compatible with most other resource packs.

## Limitations

This resource pack is **incompatible** with Sodium and may be incompatible with any mods that alter rendering.
